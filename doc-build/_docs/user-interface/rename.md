### Rename column

You can use this step to rename a column.

#### Where to find this step?

- Column header menu
- Search bar

#### Options reference

<img src="/img/docs/user-interface/rename_step_form.jpg" width="350" />

- `Old name`: the original column name

- `New name`: the new column name
